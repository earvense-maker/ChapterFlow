# ChapterFlow 0.1.0-beta.6

作品の保存先を安全に移動したり、既存のChapterFlow保存先へ切り替えたりできる公開ベータ第6版です。小説生成の文脈と表現の重複抑制も改善しました。

<img src="https://github.com/earvense-maker/ChapterFlow/releases/download/v0.1.0-beta.6/chapterflow-introduction.png" alt="ChapterFlow — AIで自分用の小説とロールプレイを作るアプリ" width="50%">

> [!IMPORTANT]
> このアプリはコード署名を行っていないため、初回起動時にWindows SmartScreenの「WindowsによってPCが保護されました」という警告が表示されることがあります。公式リリースから入手し、必要に応じて `SHA256SUMS.txt` と照合したものだけを、「詳細情報」→「実行」で起動してください。

## ダウンロード

- `ChapterFlow.Setup.0.1.0-beta.6.exe`: 通常はこちら（インストーラ版）
- `ChapterFlow.0.1.0-beta.6.exe`: インストールせずに使うポータブル版
- `SHA256SUMS.txt`: 上記exeのSHA-256確認用

対応環境は64bit版Windows 10 / 11です。自動アップデートはないため、新版はGitHub Releasesから手動で入手してください。

## beta.6の主な変更

- 設定画面から、作品データ一式を空のフォルダへ安全に移動できるようにしました。
- すでに作品が入っている別のChapterFlow保存先へ、コピーせず切り替えられるようにしました。
- 移動・切り替え中の書き込み停止、コピー内容のSHA-256検証、Junction/シンボリックリンク拒否、壊れた設定ファイルの保護を追加しました。
- 同じ保存先を別プロセスや別PCで同時利用しないためのロックを追加しました。異常終了後の古いロックは自動回収します。
- 文脈の切れ目を意識した長文取得、プロンプト長の調整、頻出表現の繰り返しを避ける指示を改善しました。
- セットアップのメモ欄と、本文同期状態の表示を改善しました。

## 利用前に確認してください

- 更新前と保存先の移動前には、作品フォルダを別の場所へバックアップしてください。
- APIキーはPC内の `config\credentials.json` に平文で保存されます。共有PCでの利用やファイル共有に注意してください。
- 生成・相談時には必要な作品情報が選択したAI事業者へ送信され、API利用料金は利用者負担です。各社の最新の料金とデータ利用条件を確認してください。
- 保存先の移動・切り替え後はアプリが再起動します。ネットワークドライブなどで他のPCが同じ保存先を使用中の場合、処理は安全のため拒否されます。
- ベータ期間中はデータ形式や画面を変更する可能性があります。不具合報告にAPIキーや公開したくない本文を添付しないでください。
