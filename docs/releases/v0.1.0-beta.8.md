# ChapterFlow 0.1.0-beta.8

本文生成の通知、生成後の自動設定レビュー、作品らしさを保ちながら場面ごとに小さな変化を加える文体変調を追加した公開ベータ第8版です。

<img src="https://github.com/earvense-maker/ChapterFlow/releases/download/v0.1.0-beta.8/chapterflow-introduction.png" alt="ChapterFlow — AIで自分用の小説とロールプレイを作るアプリ" width="50%">

> [!IMPORTANT]
> このアプリはコード署名を行っていないため、初回起動時にWindows SmartScreenの「WindowsによってPCが保護されました」という警告が表示されることがあります。公式リリースから入手し、必要に応じて `SHA256SUMS.txt` と照合したものだけを、「詳細情報」→「実行」で起動してください。

## ダウンロード

- [`ChapterFlow.Setup.0.1.0-beta.8.exe`](https://github.com/earvense-maker/ChapterFlow/releases/download/v0.1.0-beta.8/ChapterFlow.Setup.0.1.0-beta.8.exe): 通常はこちら（インストーラ版）
- [`ChapterFlow.0.1.0-beta.8.exe`](https://github.com/earvense-maker/ChapterFlow/releases/download/v0.1.0-beta.8/ChapterFlow.0.1.0-beta.8.exe): インストールせずに使うポータブル版
- [`SHA256SUMS.txt`](https://github.com/earvense-maker/ChapterFlow/releases/download/v0.1.0-beta.8/SHA256SUMS.txt): 上記exeのSHA-256確認用

対応環境は64bit版Windows 10 / 11です。自動アップデートはないため、新版はGitHub Releasesから手動で入手してください。

## beta.8の主な変更

- 本文の出力開始・生成完了・失敗を、設定に応じて通知音、システム通知、アプリ内通知で知らせられるようになりました。
- 本文生成後に世界設定・人物設定の整合性を自動走査し、提案のみ、安全な変更だけ自動適用、すべて自動適用から動作を選べるようになりました。
- 自動更新の根拠・リスク・変更前後を履歴へ残し、最新の自動更新を取り消せるようになりました。
- 文体見本、人称、視点、人物の口調を維持しながら、視覚・聴覚・身体・内省・運動・対話・時間の軸へ場面ごとの小さな傾きを加えられるようになりました。
- 書き直しでは同じ文体プロファイルを再利用し、別案では新しい傾きを選ぶことで、比較可能性と変化の両方を保ちます。
- 採用した本文だけから冒頭・着地・比喩核・反応型を分析し、直近で続いた型や感覚入口を弱く減衰します。意図的なモチーフや口癖は除外できます。
- 文体解析の失敗や破損したトレースデータは本文生成を止めず、既存の生成経路へ安全に縮退します。

## 追加のモデル利用について

- 自動設定レビューが走査を行う回は、本文生成とは別のモデル呼び出しが発生します。
- 文体変調で「直近の構成パターンを弱く減衰する」を有効にした場合、採用した本文の型を分析する追加モデル呼び出しが発生します。
- いずれも作品設定から無効化できます。API利用料金は利用者負担です。

## 利用前に確認してください

- 更新前には作品フォルダを別の場所へバックアップしてください。
- APIキーはPC内の `config\credentials.json` に平文で保存されます。共有PCでの利用やファイル共有に注意してください。
- 生成・相談・自動レビュー・文体解析時には必要な作品情報が選択したAI事業者へ送信されます。各社の最新の料金とデータ利用条件を確認してください。
- 「すべて自動適用」でも技術的な安全確認は維持されますが、世界設定や人物設定を自動更新します。履歴と確認通知を確認してください。
- ベータ期間中はデータ形式や画面を変更する可能性があります。不具合報告にAPIキーや公開したくない本文を添付しないでください。
