# ChapterFlow 0.1.0-beta.2

ChapterFlow — API-Powered Narrative Studio の公開ベータ第2版です。
初回ベータ以降に見つかった操作上の分かりにくさを改善し、作品設定と生成指示の扱いを整理しました。

## ダウンロード

- `ChapterFlow Setup 0.1.0-beta.2.exe`: 通常はこちらを使用してください。
- `ChapterFlow 0.1.0-beta.2.exe`: インストールせずに使うポータブル版です。

配布物と同じページにある `SHA256SUMS.txt` で、ダウンロードしたexeのSHA-256を確認できます。

## beta.2の主な変更

- 作品設定を「舞台・世界」と「物語の方針」に分け、設定内容を整理しやすくしました。
- 文体などのカスタム内容を、プリセットの置き換えではなく追加指示として反映するようにしました。
- 濡れ場描写の設定に「設定なし」を追加しました。
- ロールプレイの応答を停止したあと、送信した発言を訂正して送り直せるようにしました。
- 場面ページの操作、採用表示、オプション、テーマ切り替えなどの画面を整理しました。
- 設定の直接入力画面を見直し、プリセットの折りたたみとAPIキー入力位置を改善しました。
- GeminiとDeepSeekのAPIキー取得ガイドを更新しました。

## ベータ版の注意事項

- 対応環境は64bit版のWindows 10 / 11です。
- コード署名を行っていないため、Windows SmartScreenの警告が表示されます。
- 自動アップデートはありません。新版はGitHub Releasesから手動で入手してください。
- APIキーはPC内の `config\credentials.json` に平文で保存されます。
- 生成や相談では、必要な作品情報が選択したLLM事業者へ送信されます。
- API利用料金は利用者の負担です。
- OpenRouterの既定値 `google/gemma-4-31b-it:free` は無料ですが、回数制限があり、混雑や提供状況によって利用できない場合があります。その場合は作品の技術設定で別のモデル名へ変更してください。
- ベータ期間中はデータ形式や画面を変更する可能性があります。大切な作品は定期的にバックアップしてください。

詳しい手順は同梱の `利用ガイド.md` を確認してください。不具合報告ではAPIキーや公開したくない作品本文を送らないでください。
