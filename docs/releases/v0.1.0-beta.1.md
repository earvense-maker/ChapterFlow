# ChapterFlow 0.1.0-beta.1

ChapterFlow — API-Powered Narrative Studio の初回公開ベータです。
自分のAPIキーを使い、連載小説の続きを生成したり、登場人物とのロールプレイを続けたりできます。

## ダウンロード

- `ChapterFlow Setup 0.1.0-beta.1.exe`: 通常はこちらを使用してください。
- `ChapterFlow 0.1.0-beta.1.exe`: インストールせずに使うポータブル版です。

配布物と同じページにある `SHA256SUMS.txt` で、ダウンロードしたexeのSHA-256を確認できます。

## 主な機能

- 作品設定、既存本文、種メモを踏まえた章生成
- 登場人物とのロールプレイ
- 相談しながら作品設定を作るセットアップ
- システムプロンプトプリセット
- Gemini、DeepSeek、OpenAI、xAI、OpenRouterのAPI接続
- PC内への作品保存とバックアップ可能なデータ構成
- インストーラ版とポータブル版

## ベータ版の注意事項

- 対応環境は64bit版のWindows 10 / 11です。
- コード署名を行っていないため、Windows SmartScreenの警告が表示されます。
- 自動アップデートはありません。新版はGitHub Releasesから手動で入手してください。
- APIキーはPC内の `config\credentials.json` に平文で保存されます。
- 生成や相談では、必要な作品情報が選択したLLM事業者へ送信されます。
- API利用料金は利用者の負担です。
- OpenRouterの既定値 `google/gemma-4-31b-it:free` は無料ですが、回数制限があり、混雑や提供状況によって利用できない場合があります。その場合は作品の技術設定で別のモデル名へ変更してください。
- ベータ期間中はデータ形式や画面を変更する可能性があります。大切な作品は定期的にバックアップしてください。

詳しい手順は同梱の `利用ガイド.md` を確認してください。不具合報告ではAPIキーや公開したくない作品本文を送らないでください。
