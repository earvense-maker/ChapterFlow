# Yumeweaving 利用ガイド

## Yumeweaving とは

Yumeweaving は、自分専用の連載小説をローカルPC上で育てるWebアプリです。
作品設定、これまでの流れ、種メモをもとに、LLM APIを使って次の話を生成できます。
作品データはPC内に保存されます。

## 必要なもの

- Windows 10 または Windows 11
- Gemini、DeepSeek、OpenAI のいずれかの API キー

インストーラ版とポータブル版は、どちらも利用者側で Node.js をインストールする必要はありません。

## インストールと起動

### インストーラ版

1. `Yumeweaving Setup <version>.exe` を実行します。
2. SmartScreen の警告が出た場合は「詳細情報」から「実行」を選びます。
3. インストール完了後、Yumeweaving が起動します。

通常はこちらを使ってください。スタートメニューや Windows のアプリ一覧から起動・アンインストールできます。

### ポータブル版

1. `Yumeweaving <version>.exe` を、デスクトップや `Documents` などのローカルフォルダに置きます。
2. exe をダブルクリックして起動します。

インストールせずに使いたい場合はこちらを使ってください。ポータブル版も作品データは exe の横ではなく `Documents\Yumeweaving` に保存されるため、インストーラ版と同じデータを参照します。

## API キーの取得と登録

いずれか1つ以上の API キーを用意してください。

- Gemini: [Google AI Studio API keys](https://aistudio.google.com/app/apikey) に Google アカウントでログインし、API キーを作成します。
- DeepSeek: [DeepSeek API Keys](https://platform.deepseek.com/api_keys) にログインし、API キーを作成します。
- OpenAI: [OpenAI API Keys](https://platform.openai.com/api-keys) にログインし、API キーを作成します。

作成した API キーは、アプリ内の「技術設定」タブから登録します。

**生成はあなた自身のAPIキーで行われ、利用量に応じて各社から課金されます。利用前に各社の料金ページと上限設定を確認してください。**

## データの保存場所とバックアップ

作品データと設定は `Documents\Yumeweaving` に保存されます。バックアップするときは、このフォルダを外付けドライブやクラウドストレージへコピーしてください。

API キーは `Documents\Yumeweaving\config\credentials.json` に平文で保存されます。バックアップや相談のためにフォルダを人へ渡す場合は、`config` フォルダを除いてください。

インストーラ版をアンインストールしたり、ポータブル版の exe を削除したりしても、`Documents\Yumeweaving` を削除しない限り作品データは残ります。

## トラブルシューティング

### SmartScreen の警告が出る

初回配布ではコード署名を行わないため、Windows が警告を出すことがあります。入手元が正しいことを確認したうえで、「詳細情報」から「実行」を選んでください。

### アプリがすでに起動している

Yumeweaving を二重起動した場合、新しいウィンドウは開かず、既存のウィンドウが前面に表示されます。反応しない場合は、タスクマネージャーで Yumeweaving を終了してから起動し直してください。

### データが見当たらない

インストーラ版とポータブル版は同じ `Documents\Yumeweaving` を参照します。アプリ内で保存場所を変更した場合は、「設定」内のデータ保存場所を確認してください。

### 起動できない

PCを再起動してからもう一度起動してください。それでも起動しない場合は、表示されたエラーメッセージと、使っている配布物がインストーラ版かポータブル版かを控えてください。
