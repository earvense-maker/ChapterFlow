# ChapterFlow 利用ガイド

## ChapterFlow とは

ChapterFlow — API-Powered Narrative Studio は、自分のAPIキーで連載小説とロールプレイを続けるWindowsアプリです。
作品設定、これまでの流れ、種メモをもとに、LLM APIを使って次の話を生成できます。
作品データの保存先はPC内です。生成や相談を実行するときは、選択したLLM事業者のAPIへ必要な内容を送信します。

## 必要なもの

- 64bit版の Windows 10 または Windows 11
- Gemini、DeepSeek、OpenAI、xAI、OpenRouter のいずれかの API キー

初回ベータではGemini、DeepSeek、xAIを実機確認済みです。OpenAIとOpenRouterは実装と自動テストを確認済みですが、実APIでは未検証です。

インストーラ版とポータブル版は、どちらも利用者側で Node.js をインストールする必要はありません。

## インストールと起動

### インストーラ版

1. `ChapterFlow Setup <version>.exe` を実行します。
2. SmartScreen の警告が出た場合は「詳細情報」から「実行」を選びます。
3. インストール完了後、ChapterFlow が起動します。

通常はこちらを使ってください。スタートメニューや Windows のアプリ一覧から起動・アンインストールできます。


### ポータブル版

1. `ChapterFlow <version>.exe` を、デスクトップや `Documents` などのローカルフォルダに置きます。
2. exe をダブルクリックして起動します。

インストールせずに使いたい場合はこちらを使ってください。新規環境では、ポータブル版も作品データは exe の横ではなく `Documents\ChapterFlow` に保存されるため、インストーラ版と同じデータを参照します。

## API キーの取得と登録

### 初めての方へ（おすすめの始め方）

まずは **Gemini の API キーで、モデルは `gemini-3.5-flash`** から始めるのが最も手軽です。Google AI Studio では Gemini API に無料枠が用意されており、`gemini-3.5-flash` はこの無料枠の対象で、個人利用の範囲であれば無料のまま試せることが多いです。

- Google アカウントでログインして、API キーを1つ発行するだけで始められます。
- 具体的な無料枠の回数上限・条件は Google 側で予告なく変わるため、最新の内容は [Gemini API pricing](https://ai.google.dev/pricing) を確認してください。
- 使い心地を試したうえで、より重厚な文章や別モデルを試したくなったら、DeepSeek や OpenAI、xAI などに切り替えられます。

### 各社のキー発行ページ

いずれか1つ以上の API キーを用意してください。

- Gemini: [Google AI Studio API keys](https://aistudio.google.com/app/apikey) に Google アカウントでログインし、API キーを作成します。
- DeepSeek: [DeepSeek API Keys](https://platform.deepseek.com/api_keys) にログインし、API キーを作成します。
- OpenAI: [OpenAI API Keys](https://platform.openai.com/api-keys) にログインし、API キーを作成します。
- xAI: [xAI Console](https://console.x.ai/) にログインし、API キーを作成します。
- OpenRouter: [OpenRouter Keys](https://openrouter.ai/settings/keys) にログインし、APIキーを作成します。既定モデルは無料の `google/gemma-4-31b-it:free` です。無料モデルは通常20回/分・50回/日の制限があり、混雑や提供状況によって利用できない場合があります。その場合は作品の技術設定で別のモデル名へ変更してください。

作成した API キーは、アプリ内の「アプリ設定」から登録します。

**生成はあなた自身のAPIキーで行われ、利用量に応じて各社から課金されます。利用前に各社の料金ページと上限設定を確認してください。**

## データ送信とプライバシー

- 作品データはPC内に保存され、ChapterFlow独自のクラウドへは保存されません。
- 生成、相談、接続確認を実行すると、作品設定、本文、会話履歴、入力した指示のうち処理に必要な部分が、選択したLLM事業者へ送信されます。
- OpenRouterを選んだ場合、生成内容はOpenRouterと、実際に選択されたモデルの提供事業者の双方へ送信されます。`openrouter/free`では呼び出しごとに提供事業者が変わる場合があります。利用前に[OpenRouterのPrivacy設定](https://openrouter.ai/settings/privacy)と、許可するデータポリシーを確認してください。
- 送信後のデータの取り扱いは、各LLM事業者の利用規約とプライバシーポリシーに従います。
- APIキーはデータ保存先の `config\credentials.json` に平文で保存され、認証のため選択したLLM事業者へ送信されます。
- ChapterFlowには、作者独自のアクセス解析やテレメトリー送信はありません。

## データの保存場所とバックアップ

作品データと設定は `Documents\ChapterFlow` に保存されます。バックアップするときは、このフォルダを外付けドライブやクラウドストレージへコピーしてください。

Electron 版の「設定」では、現在のデータをコピーして新しい場所へ運ぶ「保存先を移動」と、すでに作品が入っている別の ChapterFlow フォルダをそのまま使う「既存の保存先に切り替える」を選べます。切り替えでは現在のデータはコピーも削除もされません。選択先の作品データと `config` 内の API キー設定が使われるため、プレビューに表示された作品を確認してから実行してください。同じ保存先を複数のPCやアプリから同時に使用しないでください。

以前の Yumeweaving に保存したデータを引き継ぐ場合は、退避したデータを `Documents\ChapterFlow` へ移してからChapterFlowを起動してください。

API キーはデータ保存先の `config\credentials.json` に平文で保存されます。バックアップや相談のためにフォルダを人へ渡す場合は、`config` フォルダを除いてください。

インストーラ版をアンインストールしたり、ポータブル版の exe を削除したりしても、データ保存先を削除しない限り作品データは残ります。

## トラブルシューティング

### SmartScreen の警告が出る

初回配布ではコード署名を行わないため、Windows が警告を出すことがあります。入手元が正しいことを確認したうえで、「詳細情報」から「実行」を選んでください。

### アプリがすでに起動している

ChapterFlow を二重起動した場合、新しいウィンドウは開かず、既存のウィンドウが前面に表示されます。反応しない場合は、タスクマネージャーで ChapterFlow を終了してから起動し直してください。

### データが見当たらない

インストーラ版とポータブル版は同じデータ保存先を参照します。アプリ内で保存場所を変更した場合は、「設定」内のデータ保存場所を確認してください。

### 起動できない

PCを再起動してからもう一度起動してください。それでも起動しない場合は、表示されたエラーメッセージと、使っている配布物がインストーラ版かポータブル版かを控えてください。
