# 受け入れテスト手順書(E6 + D6)

Electron 版(E6)と zip 版(D6)のクリーン環境受け入れテスト。1回のテストセッションで両方を検証する。所要目安: 2〜3時間。

## 0. テスト対象物(作者PC側で準備)

| 配布物 | 場所 |
|---|---|
| インストーラ | `release\electron\Yumeweaving Setup 0.1.0.exe` |
| ポータブル exe | `release\electron\Yumeweaving 0.1.0.exe` |
| zip 版 | `release\yumeweaving-v0.1.0.zip` |

3点を USB メモリまたはクラウド経由でテスト環境へ持ち込む。**`win-unpacked/` フォルダは持ち込まない**(配布物ではない)。

テスト用に使い捨ての Gemini か DeepSeek の APIキーを1つ用意しておく(生成テストに必要。テスト後に無効化してよい)。

## 1. テスト環境の用意

優先順:

1. **別の Windows PC(推奨)** — 開発ツールが入っていないもの
2. **同じPCの新規ローカルアカウント** — 設定 → アカウント → 他のユーザー → 追加(Microsoft アカウントなしのローカルアカウント)。Node は全ユーザーインストールされている場合があるため、下の事前確認で判定
3. **Windows Sandbox / VM** — Windows Pro なら「Windows サンドボックス」が手軽(ただし再起動で消えるので1セッションで完了させる)

**事前確認(これが「クリーン」の定義)**: コマンドプロンプトで以下がすべて失敗すること。

```
node -v        → 認識されない(zip版テストに必須。Electron版はNodeがあってもよい)
```

`Documents\Yumeweaving` と `%APPDATA%\Yumeweaving` が存在しないこと(存在したら削除してから開始)。

## 2. フォルダの使い方

| 用途 | 場所 | 備考 |
|---|---|---|
| zip の展開先 | `デスクトップ\yumeweaving-test\` など**ローカルフォルダ** | 共有フォルダ・ネットワークドライブ直上は不可(ガイドにもその旨記載があることを確認する) |
| ポータブル exe の置き場所 | 同上の別フォルダ | インストール版と混ざらないように分ける |
| 作品データ(自動生成) | `%USERPROFILE%\Documents\Yumeweaving` | 全形態で共通。APIキー(config\credentials.json)、LANトークン、Electronのポート記録もここ |
| Electron のウィンドウ状態 | `%APPDATA%\Yumeweaving\window-state.json` | インストール版・ポータブル版で共有される |

**注意**: 3形態すべてが同じ `Documents\Yumeweaving` を使う。**Electron 版と zip 版を同時に起動しない**こと(書き込みロックはプロセス内のみのため、同時書き込みはデータ破損の恐れ。これは既知の制限であり、テスト間は必ず前のアプリを終了してから次へ)。

各パートの間でデータを引き継ぐか消すかは各パート冒頭に記載。

| A14 | `Documents\Yumeweaving\config\server-port.json` をメモ帳で開き `{"port": 99999}` 等に書き換え → 起動 | 正常起動する(不正値を無視して取り直す)。クラッシュしない | |
| A15 | アンインストール(設定 → アプリ) | `Documents\Yumeweaving` が**消えない**。作品データが無事 | |

## 5. パートC: zip 版(D6)— Node なし状態から

パートC 開始前に `Documents\Yumeweaving` を**フォルダごと退避**(リネーム)し、まっさらな状態から開始する(初回セットアップ動線の検証のため)。

| C2 | 可能なら 7-Zip でも展開して比較 | 同上(F1 の最終確認) | |
| C3 | Node 未インストールのまま `Start-Yumeweaving.bat` を実行 | 「Node.js 20+ をインストールしてください」+ ガイド参照の案内が出て止まる(クラッシュや無言終了ではない) | |

## 7. 後片付け

1. アプリのアンインストール(パートAで実施済みならスキップ)、ポータブル exe と zip 展開フォルダの削除
2. `Documents\Yumeweaving` の削除(退避したフォルダも)と `%APPDATA%\Yumeweaving` の削除
3. テスト用 APIキーの無効化(各社のコンソールから)
4. テスト用ローカルアカウントを使った場合はアカウント削除
