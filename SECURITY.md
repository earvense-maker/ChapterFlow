# セキュリティポリシー

## サポート対象

原則として、GitHub Releasesで公開している最新バージョンを対象にセキュリティ修正を行います。

## 脆弱性の報告

第三者に悪用される可能性がある問題は、公開Issueではなく
[GitHub Security Advisory](https://github.com/earvense-maker/ChapterFlow/security/advisories/new)
から非公開で報告してください。APIキー、作品本文、個人情報は報告へ含めないでください。

通常の不具合や機能要望は[Issues](https://github.com/earvense-maker/ChapterFlow/issues)へお願いします。

## 利用上の注意

- APIキーは利用者のPC内に平文で保存されます。
- 生成や相談では、処理に必要な作品情報が選択したLLM事業者へ送信されます。
- LAN配信は作者・開発者向け機能です。信頼できるプライベートネットワークでのみ使用してください。
